If you are using IntelliJ IDEA and want to make it a bit easier to conform to our coding standards, you can import
the settings files by selecting File -> Manage IDE Settings -> Import Settings and then navigate to and select
idea_settings.jar in the megamek\idea folder.  Once you've imported the settings, do the following:
1. Select File -> Settings -> Editor -> Code Style and select mm_code_style for the scheme.  
2. elect Inspections and choose MegaMek for your profile.  
3. Finally, go to Copyright and select MegaMek as your Default Project Copyright. 
    (note, Copyright is not available yet as of January 7, 2022)
